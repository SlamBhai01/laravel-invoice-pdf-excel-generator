# ‚úÖ **Problem #1 ‚Äî form_title ko array samjha gaya**

### ‚ùå Aap ka code:

```php
foreach ($form_data['form_title'] as $form_title)
```

### ‚ùó Issue:

`form_title` ek **string** hai, array nahi. JSON me:

```json
"form_title": "Employee Form"
```

**String per foreach lagayenge to error aayega.**

### ‚úî Simple Fix:

```php
$title = $this->dynamic_translate($form_data['form_title'], 'ur');
```

---

# ‚úÖ **Problem #2 ‚Äî fields ko galat tarah loop kiya**

### ‚ùå Aap ka code:

```php
foreach ($form_data['fields']['label'] as $field)
```

### ‚ùó Issue:

`fields` ek **array of objects** hai, single label array nahi.

JSON me:

```json
"fields": [
  { "label": "Full Name", ... },
  { "label": "Email", ... }
]
```

### ‚úî Correct:

```php
foreach ($form_data['fields'] as $field)
```

# ‚úÖ **Problem #3 ‚Äî name ki jaga key use kiya**

### ‚ùå Aap ka code:

```php
name='{$field['key']}'
```

### ‚ùó Issue:

JSON me **key naam ka field exist hi nahi**:

JSON me:

```json
"name": "full_name"
```

Aapne ‡§ó‡§≤‡§§ name use kiya the.

### ‚úî Fix:

```php

```

# ‚úÖ **Problem #4 ‚Äî textarea type handle nahi kiya**

### ‚ùå Aap ka code:

Textarea koi handling nahi.

JSON me:

```json
{ "type": "textarea", "name": "address" }
```

### ‚ùó Is liye error:

Controller input tag banane ki koshish karta hai, jab type `textarea` ho.

### ‚úî Fix Code:

```php
if ($field['type'] == 'textarea') {
    $html .= "<textarea name='{$field['name']}'></textarea><br><br>";
    continue;
}
```
