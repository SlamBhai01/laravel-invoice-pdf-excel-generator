# Laravel Invoice PDF & Excel Generator

This is a **Laravel 12** project that allows generating invoices in **PDF** and **Excel** formats using **custom JSON data**.

---

## Features

- Generate invoices in **PDF** format.
- Generate invoices in **Excel (.xls)** format.
- Supports **Urdu and English** content.
- Uses **Blade templates** to design the invoice layout.
- Can use **custom JSON data** for invoice items, customer info, and business details.
- Stores generated files in `storage/app/public/results/`.
